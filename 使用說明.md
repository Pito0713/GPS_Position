## 3. 啟動專案

### ⚠️ 重要：必須用 localhost 開啟前端

**不能**用 `open index.html`（file:// 協議無法取得定位權限）

```bash
# 後端啟動後，用瀏覽器開啟：
http://localhost:7788/ui
```

---

### 實機模式（iPhone）

```bash
cd ~/gps-spoofer
sudo /Applications/Xcode.app/Contents/Developer/usr/bin/python3 gps_spoofer.py
```

然後用瀏覽器開啟 `http://localhost:7788/ui`，點「**▶ 啟動通道**」。

---

### Simulator 模式（Xcode）

```bash
# Step 1：確認 Simulator 已開啟
xcrun simctl list devices | grep Booted

# Step 2：啟動後端（不需要 sudo）
python3 gps_spoofer_sim.py
```

然後用瀏覽器開啟 `http://localhost:7788/ui`，點「**⟳ 確認 Simulator**」。

## 4. 功能說明

### 📍 傳送（GOTO）

瞬間跳到指定座標。

- 地圖上點選 → 座標自動填入 → 點「**▶ 設定此位置**」
- 搜尋框輸入地名（Enter）
- 手動輸入緯度 / 經度
- 快速城市：8 個預設城市

**還原真實位置：** 點「**✕ 清除 — 還原真實 GPS**」

---

### ❄️ 冷卻計時

傳送後自動觸發冷卻倒數，防止遊戲 App 偵測到急速位移。

**觸發時機：**
- 開啟網頁時自動取得真實位置作為冷卻基準
- 第一次虛擬傳送立刻計算距離並啟動冷卻
- 之後每次傳送都會重新計算

**冷卻計算（依距離）：**

| 距離 | 冷卻時間 |
|------|---------|
| < 100m | 不觸發 |
| 1 km | 約 18 分鐘 |
| 3 km | 30 分鐘（上限） |

**倒數期間：** 「▶ 設定此位置」按鈕顯示「▶ 冷卻中 2:30 · 3.2km」並禁用。

**持久化：** 冷卻狀態存於 localStorage，重整瀏覽器後自動恢復剩餘倒數。

**模式切換：** 可在冷卻模式下拉選單切換「依距離自動計算」或「關閉冷卻」。

---

### 💾 儲存常用位置

1. 填入座標後輸入名稱（留空自動命名）
2. 點「**儲存**」
3. 點清單名稱 = 載入，點「✕」= 刪除

儲存於 localStorage，關閉後保留。

---

### 🗺 路線移動（ROUTE）

沿設定路線連續移動。

**設定步驟：**
1. 切換「路線」頁籤
2. 選擇路線模式（手動 / 步行路線 / 騎車路線）
3. 點地圖右上角「**✚**」後在地圖點選路線點
4. 選擇速度、循環次數 → 點「**▶ 開始**」

**路線模式：**

| 模式 | 說明 |
|------|------|
| ✚ 手動點選 | 直線連接各點 |
| 🚶 步行路線 | 沿真實步行道路（需 ORS API Key） |
| 🚲 騎車路線 | 沿真實腳踏車道路（需 ORS API Key） |

**ORS API Key 取得：** `openrouteservice.org` → 免費註冊 → Dashboard → Tokens

**速度快速按鈕：**

| 按鈕 | 速度 |
|------|------|
| 🚶 步行 | 5 km/h |
| 🚶‍♂️ 快走 | 8 km/h |
| 🚲 騎車 | 15 km/h |

**Multi-Stop 停留：** 每個路線點可設定停留秒數。

**中途改速：** 播放中拖動「即時速度」滑桿 → 點「套用」。

**批次輸入座標：**

```
34.52304997, 134.86966945
34.52304997, 134.99966945
34.60304997, 134.99966945
```

支援逗號、空格分隔，每行一點，點「**▶ 匯入座標**」自動加入路線。

---

### 📂 GPX 匯入

選擇 `.gpx` 檔案 → 路線預覽在地圖 → 設定循環次數 → 點「**▶ 播放**」。

---

### 🎲 隨機漫步（WALK）

在指定範圍內自然閒逛，方向帶慣性，靠近邊界自動拉回。

1. 先傳送到目標中心位置
2. 切換「漫步」頁籤
3. 設定半徑（50–5000m）、速度、循環次數
4. 點「**▶ 開始漫步**」

---

### ⌨️ 鍵盤控制

1. 點「**啟用鍵盤**」
2. 使用 `↑↓←→` 或 `WASD` 控制方向
3. 單次按下 = 移動一步，按住 = 持續移動
4. 步長滑桿調整每步距離（1–100m，預設 5m）

在 input 輸入框內時不觸發鍵盤控制。

---

### 路線通用控制

| 按鈕 | 功能 |
|------|------|
| ▶ 開始 | 開始路線 / 漫步 |
| ⏸ 暫停 | 停在當前位置 |
| ■ 停止 | 停止，**位置保持在最後一點** |
| ✕ 清除 | 還原真實 GPS |

> **停止 ≠ 清除。** 停止後 iPhone 仍在最後虛擬位置，需點「清除」才還原真實 GPS。

## 5. 常見問題

**真實位置無法顯示**
→ 必須用 `http://localhost:7788/ui` 開啟，不能用 `file://`

**Tunnel 連線失敗**
→ iPhone 螢幕解鎖、USB 插好、已點「信任此電腦」

**傳送成功但 iPhone 沒變**
→ 先執行「掛載 DDI」，重啟 iPhone 地圖 App

**路線走幾步就停**
→ Tunnel 斷線，重新點「啟動通道」

**停止後跳回真實位置**
→ 後端被重啟，process pool 清空。重新傳送一個位置鎖定即可

**Simulator 傳送失敗**
→ 確認 Simulator 已開啟且裝置 Booted：`xcrun simctl list devices | grep Booted`

**iOS 更新後無法使用**
→ 重新點「掛載 DDI」

## 6. 安全說明

所有通訊在本機進行，不連接外部伺服器。

| 元件 | 說明 |
|------|------|
| 後端 API | 只綁定 127.0.0.1 |
| 地圖圖層 | OpenStreetMap CDN（會看到你的 IP） |
| 地名搜尋 | Nominatim（搜尋關鍵字會送出） |
| 道路路線 | OpenRouteService API（路線座標會送出） |
| 儲存位置 | localStorage，只存本機 |
| 冷卻狀態 | localStorage，只存本機 |
